# 均线回调策略 - 完整文档

> 回测周期：最近120个交易日 | 数据源：天勤TqSdk | 回测日期：2026-04-02

---

## 一、策略核心逻辑

趋势中等待价格回调到均线后重新站回，顺势入场。

**一句话总结：10分钟图上，趋势确认后，价格跌破10MA再收回来，且回调持续了至少4根K线——同时开两笔单，大仓位吃短利润，小仓位博大波段。**

---

## 二、入场条件（两个策略共用）

| 序号 | 条件 | 说明 |
|------|------|------|
| 1 | **周期：10分钟** | 三品种验证最稳定的周期 |
| 2 | **趋势确认：10MA > 20MA（做多）** | 反之做空。均线金叉/死叉判断方向 |
| 3 | **B类信号** | 价格收盘跌破10MA → 之后某根K线**收盘重新站回10MA上方** → 该K线收盘价入场 |
| 4 | **回调K线 >= 4根** | 从跌破10MA到收回10MA之间，至少经过4根K线（充分回调） |
| 5 | **初始止损 = 本次回调最低点 - 5跳** | 入场时立即挂好止损单 |

### 做空反过来
- 10MA < 20MA
- 价格收盘突破10MA上方 → 重新跌回10MA下方 → 入场做空
- 初始止损 = 回调最高点 + 5跳

---

## 三、两个策略 / 两种出场

一个信号触发时，**同时开两笔单**：

### S1 快刀（大仓位）—— 高胜率，快进快出

| 项目 | 内容 |
|------|------|
| **出场方式** | 新高K线追踪止损（exit_newhigh） |
| **止损逻辑** | 入场后出现实体新高K线 → 止损提高到该K线最低价 - 5跳（只升不降） |
| **特点** | 赚了快跑，平均持仓约50分钟 |
| **适合仓位** | 大仓位（如每笔风险2%账户资金） |

### S2 波段（小仓位）—— 高盈亏比，博大波段

| 项目 | 内容 |
|------|------|
| **出场方式** | 回调低点追踪止损（exit_pullback） |
| **止损逻辑** | 持仓期间价格形成新的回调 → 回调结束后 → 止损更新到本次回调最低点（只升不降） |
| **特点** | 大部分时候小亏，赢一次顶多次亏损，平均持仓约3小时 |
| **适合仓位** | 小仓位（如每笔风险1%账户资金） |

|------|--------|--------|------|--------|----------|----------|----------|------|

### 结论
- **白银、黄金、原油：策略有效**，两种出场方式EV均显著为正
- **LPG：策略无效**，EV接近零，不建议用此策略交易LPG
- 策略对**贵金属和能化龙头品种**有效，并非所有品种通用

## 六、仓位管理建议（以10万账户为例）

| | S1 快刀 | S2 波段 |
|---|---|---|
| 每笔风险上限 | 2%账户 = 2000元 | 1%账户 = 1000元 |
| 仓位计算 | 2000 / (止损距离 x 合约乘数) = 手数 | 1000 / (止损距离 x 合约乘数) = 手数 |
| 最惨连亏损失 | ~7笔 x 2% ≈ 不会同时全亏，实际约3-4% | ~13笔 x 1% ≈ 实际约5-7% |

**做期权的天然优势**：权利金就是最大亏损，不存在滑点风险。
- S1：买平值/浅实值期权（持仓50分钟，Theta衰减小）
- S2：买轻度虚值期权（小仓位，赢了Gamma加速放大利润）

---

## 七、期望值公式

```
期望值(EV) = 胜率 × 盈亏比 - (1 - 胜率)
```

- EV > 0 = 长期能赚钱
- EV > 0.5 = 优秀策略
- EV > 1.0 = 极优策略

以S1白银为例：42.9% × 3.53 - (1 - 42.9%) = +0.94

## 八、验证过程中排除的条件

以下条件在回测中被验证为**不可靠**（跨品种不一致）：

| 条件 | 问题 |
|------|------|
| A类信号（影线碰10MA弹回） | 部分品种/周期EV为负 |
| 均线宽度 >= 0.5% | 黄金上为负 |
| 距100MA 1-2倍ATR | 白银上为负 |
| 60分钟周期 | 原油上为负 |
| exit_ma10（破10MA出场） | 多品种EV接近零 |
| exit_ma20（破20MA出场） | 原油上大幅为负 |

## 十一、关键提醒

1. **回测 ≠ 实盘**。120天数据验证了策略逻辑合理，但不保证未来收益
2. **不是所有品种都适用**。LPG已证明无效，新品种必须先验证
3. **连亏是正常的**。40%胜率意味着经常连亏5-7笔，但每笔亏损很小
4. **严格止损是核心**。这个策略的优势就是止损紧（平均亏损0.1-0.3%），绝对不能扛单
5. **两个策略互补**。S1贡献稳定小利润，S2博取偶尔的大波段
